# AI 文本情感分析系统

## 学生信息
- 姓名：你的姓名
- 学号：你的学号

## 项目简介
基于 DistilBERT 预训练模型的轻量级文本情感分析系统，可快速判断输入文本的情感倾向（正面/负面）。

## 核心功能
- ✅ 前端页面展示个人姓名与学号
- ✅ AI 情感分析：输入文本 → 输出情感标签 + 置信度
- ✅ 可迁移：依赖简单，支持本地/云端部署
- ✅ 可优化：可更换模型、新增多语言支持、优化界面
- ✅ 可调参：可修改模型类型、输入长度、批处理大小等

## 运行方式
1.  安装依赖：`pip install -r requirements.txt`
2.  启动服务：`python app.py`
3.  访问：`http://127.0.0.1:5000`
